What Is The Event Log?

The Event Log allows you to view and monitor the activities performed on your account. You can access the Event Log by navigating to Account > Audit Log in $cpanelnamefirstuse.

To search for particular records, specify search item, set the Date Range and click the Search button. 

  • Basic Search
  • Advanced Search

The Event Log Basic Search shows the following filters: 

  • Date: date and time when the action took place (shown in Pacific Standard Time).
  • Login: the Account Contact who triggered the event.
  • Action: description of the action performed (Create, Install, Modify, Send Notification, Uninstall, etc).
  • Item: the service that was changed.
  • Affected: the specific item or user that was changed.
  • Description: details of the event describing the nature of changes, or an error message.

Under Advanced Search option you can use the following filters to narrow down your search results:

  • Description: specify the term to search for across all columns.
  • Action: select the type of event from the drop-down.
  • Item: select the service that was affected.
  • Login: type in the login of an Account Contact. You can use the auto-suggest feature by typing in the first two letters of the login.

Notes:

  • Items in red indicate errors that have occurred. These can denote a variety of different errors (temporary errors that were resolved almost immediately, user errors, system errors, browser errors, etc.) and most likely have not caused issues with your account. If you do believe there is an issue with your account, please contact support.
  • You can select the Errors only checkbox if you would like to exclude all events that do not contain errors.
  • If you have more than a total of 25 records to display, a button appears at the bottom of the log allowing you to see the next 25 records. You can also select how many events are displayed on the page (10, 25, 50 or 100).
  • To export the entire log, click the Export to CSV File button at the top-right corner. If you are exporting a large log, it may take a few minutes to generate the file and download it.
  • You can see the IP address used to log in as an Account Contact who initiated the event by clicking on + icon.
